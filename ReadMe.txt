📦 画像移動ツール「ImageShuffler」仕様書（Codex向け）
1. 概要
本ツールは、複数の画像フォルダ（対象フォルダ）から、指定された枚数の画像を選び、1つの移動先フォルダへ移動するGUIアプリケーションです。
移動順序は「ファイル名順」または「ランダム」から選択でき、各フォルダごとに移動枚数を個別に設定できます。
さらに、登録した対象フォルダをいつでも「取り消しボタン」で削除できる、操作しやすいUIが特徴です。

2. 主な機能
2.1 対象フォルダ登録
ユーザーは複数のフォルダを「対象フォルダ」として登録できる

登録された各フォルダについて：

フォルダパス表示

移動可能枚数（対象画像数）を自動カウントして表示

移動枚数を手動で入力（上限は移動可能枚数）

取り消しボタンでリストから行単位で削除可能

2.2 移動先フォルダ選択
1つの移動先フォルダを指定可能（変更・再選択も可能）

対象フォルダと同じパスは選択不可（UI上で制御）

2.3 並び順指定（全体共通設定）
「ファイル名順（昇順）」または「ランダム」のいずれかを選択

すべての対象フォルダに一括適用

2.4 実行処理
実行時、対象フォルダごとに以下の動作を行う：

指定枚数の画像を選出（順序に従う）

移動先フォルダへ移動（cut）

同名ファイルがある場合はリネーム（例：file_1.jpg）

各フォルダ内の画像が指定数に満たない場合は、移動可能な分だけ処理し、警告ログに記録

2.5 ログと進捗
実行後に処理結果をUIログエリアに表示（成功/失敗/スキップの内訳など）

任意でログファイル（.txt）として保存可能

3. 対象ファイル条件
対象画像拡張子：.jpg, .jpeg, .png, .webp（大文字小文字問わず）

対象フォルダ直下のみをスキャン（サブフォルダは対象外）

4. UI構成
4.1 対象フォルダ一覧テーブル
フォルダパス	移動可能枚数	移動枚数（入力欄）	取り消し
C:\images\A	12枚	[ 5 ]	🔄 取り消し
C:\images\B	3枚	[ 3 ]	🔄 取り消し

行ごとの「取り消しボタン」で対象フォルダを削除可能

一覧の下部に「合計移動枚数」をリアルタイム表示

4.2 操作エリア
📂 「対象フォルダを追加」ボタン（フォルダ選択ダイアログ）

📁 「移動先フォルダを選択」ボタン（フォルダ選択ダイアログ）

🔃 並び順：ラジオボタン（ファイル名順 / ランダム）

▶ 実行ボタン

📝 ログ表示エリア（スクロール対応）
